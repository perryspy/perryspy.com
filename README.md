# [perryspy.com](https://perryspy.com)

Personal portfolio site built with React + Vite.

## Dev

```bash
npm install
npm run dev      # localhost:5173
```

## Deploy

```bash
npm run build    # outputs to docs/
```

Commit and push `docs/` — GitHub Pages serves from there.

## Editing

- **Content** — `src/content.js`
- **Styles** — `src/App.css` (CSS variables at the top)
- **Layout/markup** — `src/components/`
